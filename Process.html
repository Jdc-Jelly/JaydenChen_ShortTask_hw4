PhoneGap

PhoneGap is a web-based framework that uses web technologies such as HTML, CSS, and JavaScript to build cross-platform mobile applications with device API access and fast iteration through live reloading. Phonegap uses WebView as its main component from native library.

  • Install Node.js, PhoneGap CLI, and a code editor of your choice. Also, set up an emulator or physical device for testing.
  • Create a new PhoneGap project: Use the PhoneGap CLI to create a new project with the desired settings and features.
  • Develop the user interface: Use web technologies such as HTML, CSS, and JavaScript to create the app's user interface. PhoneGap provides access to device APIs and features, so developers can write code that interacts with device sensors, camera, and other hardware.
  • Write business logic: Use JavaScript programming language to write the application's business logic.
  • Use PhoneGap's built-in testing framework or a third-party testing library to write unit, integration, and functional tests to ensure that the app works as expected.

Reactive Native

React Native is a JavaScript framework that uses a single codebase to build native mobile applications for iOS and Android with platform-specific UI components and fast iteration through hot reloading. React Native maps its Tags with native components of Android and iOS.

  • Install Node.js, React Native CLI, and a code editor of your choice.
  • Use the React Native CLI to create a new project with the desired settings and features.
  • Use React's component-based architecture to create reusable, composable UI elements that can be used to create the app's user interface.
  • Use JavaScript programming language to write the application's business logic. React Native provides access to native APIs and features, so developers can write code that interacts with device sensors, camera, and other hardware.
  • Use React Native's built-in testing framework or a third-party testing library to write unit, integration, and functional tests to ensure that the app works as expected.

Flutter

Flutter is a cross-platform framework that uses a single codebase to build native mobile, web, and desktop applications with fast development cycles and customizable UI widgets.Flutter draw its own components (widgets) on Canvas from the native library.

  • Set up development environment: Install the Flutter SDK and a code editor of your choice.
  • Use the Flutter command-line interface to create a new project with the desired settings and features.
  • Flutter's UI toolkit, called widgets, allows developers to create beautiful, highly-customizable user interfaces. Widgets are used to create visual elements like buttons, text fields, and images.
  • Use Dart programming language to write the application. Dart is a modern, object-oriented language that's easy to learn and use.
  • Use Flutter's built-in testing framework to write unit, integration, and functional tests to ensure that the app works as expected.
